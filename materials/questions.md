# Вопросы для подготовки к собеседованию
## junior
### Общие вопросы

1. Основные принципы ООП
<details> <summary>Ответ</summary> <p>
    1. Инкапсуляция - получение данных через публичные методы<br>
    2. Наследование - создание новых классов на основе существующих<br>
    3. Полиморфизм - использование одинакового интерфейса<br>
    4. Абстракция - позволяет выделить общие характеристики объекта<br>
</p> </details>

2. Сложность алгоритма
<details>
    <summary>Ответ</summary>
        <p>
        Позволяет описать сколько времени и ресурсов потребует программа для достижения цели
        </p>
</details>

3. Стек и очередь
<details>
    <summary>Ответ</summary>
        <p>
        1. Стек - абстрактный тип данных, представляющий собой список элементов, организованных по принципу LIFO (англ. last in — first out, «последним пришёл — первым вышел»)<br>
        2. Очередь - последний пришел последний вышел<br>
        </p>
</details>

4. Что такое шаблонный класс или функци
<details>
    <summary>Ответ</summary>
        <p>
            1. Обобщенный класс позволяет работать с разными типами данных ключевое слово template<br>
            <pre>
	            <code>
                template<typename Type>
                Type max(Type a, Type b){ 
                    return (a >= b ? a : b);
                }
	            </code>
            </pre>

            2. шаблонная функция 
    
            <pre>
	            <code>
                template<typename Type>
                Type max(Type a, Type b){ 
                    return (a >= b ? a : b);
                }
	            </code>
            </pre>
        </p>
</details>
